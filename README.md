## About

Easy games for improving your brain power.

## Installation

```
pip install --index-url https://test.pypi.org/simple/ --extra-index-url https://pypi.org/simple --no-cache-dir sscream-brain-games
```

## Available commands

* brain-games
* brain-calc
* brain-even
* brain-gcd
* brain-prime
* brain-progression
